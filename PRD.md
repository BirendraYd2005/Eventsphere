# Product Requirement Document (PRD)

## Project Name
EventSphere

## Problem Statement

Many event organizers struggle to efficiently manage registrations and promote their events, while users often find it difficult to discover events in one place.

## Solution

EventSphere provides a centralized platform where organizers can create and manage events, and users can browse, register, and receive updates about events.

## Target Users

- Students
- Event Organizers
- Colleges
- Companies
- General Public

## Core Features

- User Registration & Login
- Browse Events
- Search & Filter Events
- Event Creation
- Event Booking
- Organizer Dashboard
- Admin Dashboard

## Technology Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Future Enhancements

- Online Payment Integration
- QR Code Tickets
- AI Event Recommendations
- Email Notifications
